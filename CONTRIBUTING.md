# Contributing

## This is not an open source project

Arcane Scout is **source-available**, not open source. The source code is published for transparency and reference only. Pull requests and code contributions are not accepted. See [LICENSE](LICENSE) for the full terms.

---

## Bug reports & feature requests

You can open a GitHub issue to:

- Report a bug (please include steps to reproduce, expected vs. actual behaviour, and your Chrome version)
- Request a new feature or improvement

---

## Proposing ideas by email

If you have a more detailed proposal or want to discuss a feature before opening an issue, you can reach me at [mateo@callec.net](mailto:mateo@callec.net). Please note that code patches submitted by email will not be reviewed or applied.

---

## Security issues

Do not open a public issue for security vulnerabilities. See [SECURITY.md](SECURITY.md) for the responsible disclosure process.
