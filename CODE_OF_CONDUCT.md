# Code of Conduct

## Our Standards

When interacting with this project — through issues, emails, or any other channel — please:

- Be respectful and constructive in feedback and discussion
- Stay on topic (bugs, features, security)
- Accept that feature requests and proposals may not be implemented

The following are not acceptable:

- Harassment, insults, or personal attacks of any kind
- Spam or off-topic content
- Public disclosure of security vulnerabilities (see [SECURITY.md](SECURITY.md))

## Enforcement

Interactions that violate these standards may result in issue closure or being blocked from future communication with this project.
